#+TITLE: Bookmark+ Command Reference
#+OPTIONS: toc:2 num:nil
#+STARTUP: showall

This is a curated list of the user-level commands you are most
likely to want.  Internal / less common commands are omitted; you
can still discover them with =M-x bmkp-= =TAB=.

* Creating bookmarks

| Command                                    | Default key  | What it does                                |
|--------------------------------------------+--------------+---------------------------------------------|
| =bmkp-bookmark-set-confirm-overwrite=        | =C-x r m=, =C-x x m=, =C-x x c m= | Set / update a bookmark.  Buffer-local default. |
| =bookmark-set=                              | =C-x x c M=    | Vanilla =bookmark-set=.                       |
| =bmkp-toggle-autonamed-bookmark-set/delete=  | =C-x x RET=, =C-x x c RET= | Toggle autonamed bookmark here.             |
| =bmkp-autofile-set= (=bmkp-bookmark-a-file=)  | =C-x x c a=    | Autofile bookmark for a file.                |
| =bmkp-file-target-set=                       | =C-x x c f=    | File bookmark (file at point).                |
| =bmkp-url-target-set=                        | =C-x x c u=    | URL bookmark.                                 |
| =bmkp-set-desktop-bookmark=                  | =C-x x K=, =C-x x c K=, =C-x r K= | Desktop bookmark.                           |
| =bmkp-set-sequence-bookmark=                 | =C-x x c s=    | Sequence (composite) bookmark.                |
| =bmkp-make-function-bookmark=                | =C-x x c F=    | Function (or last-kbd-macro) bookmark.        |
| =bmkp-wrap-bookmark-with-last-kbd-macro=     | =C-x x c C-k=  | Sequence: bookmark + last kbd macro.          |
| =bmkp-set-bookmark-file-bookmark=            | =C-x x y=, =C-x x c y= | Bookmark-file bookmark.                     |
| =bmkp-set-snippet-bookmark=                  | =C-x x M-w=, =C-x x c M-w= | Snippet bookmark from active region.        |
| =bmkp-set-autonamed-bookmark=                | --           | Plain autonamed bookmark at point.            |
| =bmkp-set-autonamed-bookmark-at-line=        | --           | Autonamed bookmark at start of line.          |
| =bmkp-set-autonamed-regexp-buffer=           | --           | One per regexp match in buffer.               |
| =bmkp-set-autonamed-regexp-region=           | --           | One per regexp match in region.               |
| =bmkp-set-variable-list-bookmark=            | --           | Bookmark a set of variables and values.       |
| =bmkp-set-izones-bookmark=                   | --           | Bookmark =zz-izones= (with =zones.el=).        |
| =bmkp-set-kmacro-bookmark=                   | --           | Bookmark a named keyboard macro.              |
| =bmkp-set-kmacro-list-bookmark=              | --           | Bookmark the entire keyboard-macro ring.      |
| =bmkp-set-automatic-bookmark=                | --           | Put on a hook for automatic bookmarking.      |
| =bmkp-compilation-target-set=                | =C-c C-b= (compilation, grep) | Bookmark the hit at point.                    |
| =bmkp-compilation-target-set-all=            | =C-c C-M-b=    | Bookmark every hit.                           |
| =bmkp-occur-target-set=                      | =C-c C-b= (Occur) | Bookmark the hit at point.                    |
| =bmkp-occur-target-set-all=                  | =C-c C-M-b=    | Bookmark every hit.                           |
| =bmkp-occur-create-autonamed-bookmarks=      | =C-c C-M-B=    | Autonamed bookmark per Occur hit.             |
| =bmkp-clone-bookmark=                        | =C-x x 2=      | Duplicate a bookmark.                         |
| =bmkp-empty-file=                            | =C-x x 0=      | Create / empty a bookmark file.               |
| =bookmark-write=                             | =C-x x w=      | Write current bookmarks to a chosen file.     |
